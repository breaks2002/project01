# 数据导入导出功能说明

## 📥 JSON 导入导出用途

JSON 导入导出用于完整的模型备份和分享：

| 用途 | 说明 |
|------|------|
| **完整模型备份** | 保存所有节点的完整状态（位置、格式、公式、时间序列数据等） |
| **模型分享** | 将完整的业务模型分享给其他用户 |
| **版本管理** | 保存不同版本的模型进行对比 |
| **完整恢复** | 导入时可以 100% 恢复之前的工作状态 |

## 📊 Excel 导入功能

### 支持的格式
- **.xlsx** - Excel 2007+ 工作簿（推荐）
- **.xls** - Excel 97-2003 工作簿
- **.csv** - CSV 逗号分隔文本文件（原有功能）

### 使用方法

1. 点击工具栏的 **"导入数据"** 按钮
2. 选择你的 Excel 文件（.xlsx 或 .xls）
3. 系统会自动检测是数据表还是公式表
4. 按照向导完成导入

### 两种导入模式

#### 1️⃣ 数据表模式
用于导入时间序列数据，列结构示例：

| 指标名称 | 1月实际 | 2月实际 | ... | 9月预测 | ... |
|---------|---------|---------|-----|---------|-----|
| 营业收入 | 100000 | 110000 | ... | ... | ... |
| 变动成本 | 50000 | 55000 | ... | ... | ... |

#### 2️⃣ 公式表模式
用于导入节点属性和公式，列结构示例：

| 指标ID | 指标名称 | 节点类型 | 公式 | 最小值 | 最大值 | ... |
|--------|---------|---------|------|--------|--------|-----|
| yingyeshouru | 营业收入 | driver | | 0 | 100000 | ... |
| maolirun | 毛利润 | computed | yingyeshouru - yingyechengben | | | ... |

## 📝 如何创建 Excel 测试文件

由于 Excel 文件是二进制格式，你可以通过以下方式创建：

### 方法一：从 CSV 转换（推荐）
1. 打开现有的 `利润表测试数据.csv`
2. 在 Excel 中打开，然后另存为 `.xlsx` 格式

### 方法二：使用 Node.js 脚本生成
项目包含 `create-excel-files.js` 脚本，可以批量生成 Excel 文件：

```bash
# 运行脚本（需要先安装依赖）
node create-excel-files.js
```
