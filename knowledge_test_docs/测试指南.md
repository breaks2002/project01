# AI 智能调参 - 知识库功能测试指南

## 📋 测试目标

验证知识库模块的核心功能：
1. 文档上传与解析
2. 语义检索与相似度匹配
3. 知识库与 AI 调参流程整合
4. 历史案例对 AI 建议的影响

---

## 🔧 测试前准备

### 1. 确认测试文件位置
测试文档已创建在以下目录：
```
E:\MY AI\ValQ\knowledge_test_docs\
├── 制造业降本增效方案.md
├── 零售电商 Q4 旺季营销方案.md
```

- 点击 Toolbar「AI 决策」→「AI 配置」
- 确保已配置 API URL 和 API Key

## 📝 测试步骤

### 步骤 1：上传测试文档到知识库

1. 打开应用，点击 Toolbar 的「AI 决策」→「知识库」
2. 点击「➕ 上传文档」按钮
3. 选择文件：`制造业降本增效方案.md`
4. 填写元数据（可选）：
   - 标题：制造业降本增效方案
   - 行业：制造业
   - 标签：降本增效、成本优化、利润提升
5. 点击「确认上传」
6. ✅ **验证点**：文档显示在知识库列表中

**重复上述步骤上传另外两个文档**

---

### 步骤 2：测试知识库检索功能

1. 在知识库面板顶部的搜索框输入：`降本`
2. 点击「搜索」按钮
3. ✅ **验证点**：
   - 应该返回「制造业降本增效方案」
   - 显示相似度百分比（如 85%）
   - 制造业文档排在第一位

4. 清空搜索框，输入：`双 11`
5. ✅ **验证点**：
   - 应该返回「零售电商 Q4 旺季营销方案」
   - 相似度应该在 80% 以上

---

### 步骤 3：测试 AI 调参 - 知识库集成

1. 关闭知识库面板
2. 点击 Toolbar「AI 决策」→「AI 调参」
3. 在业务背景输入框中输入以下测试用例：

**测试用例 A：制造业场景**
```
我是一家制造企业，面临原材料成本上涨压力，净利润率从 12% 下降到 8%。
希望通过优化成本结构和提升运营效率，实现净利润率回升到 15% 的目标。
营业成本目前占比 50%，希望能优化到 45% 左右。
```

4. 点击「开始智能分析」
5. 等待 AI 分析完成（约 10-30 秒）
6. ✅ **验证点**：
   - AI 建议中应该包含「营业成本」的调整
   - 调整方向应该是「降低」或「优化」
   - 建议理由应该提及「采购优化」「工艺优化」「自动化」等关键词
   - 与知识库中的制造业案例建议相似

---

**测试用例 B：零售电商场景**
```
我们是品牌电商企业，Q4 是销售旺季，想抓住双 11 双 12 机会提升业绩。
目前月销售额约 300 万，希望 Q4 能提升到 550 万左右。
销售费用可以适当增加，但要控制费用率在 20% 以内。
```

7. 清空输入框，输入上述测试用例 B
8. 点击「开始智能分析」
9. ✅ **验证点**：
   - AI 建议中应该包含「营业收入」大幅提升
   - 销售费用同步增长但费用率可控
   - 提及「双 11」「双 12」「旺季」等关键词
   - 月度分配策略显示 Q4 重点投入

---

**测试用例 C：科技研发场景**
```
公司是 SaaS 企业，正处于成长期，需要加大研发投入提升产品竞争力。
年收入 8000 万，希望增长到 1.2 亿。
研发费用想从 1600 万增加到 2400 万，同时实现扭亏为盈。
```

10. 清空输入框，输入上述测试用例 C
11. 点击「开始智能分析」
12. ✅ **验证点**：
    - AI 建议中应该包含「研发费用」增加
    - 同时有「营业收入」增长建议
    - 提及「SaaS」「ARR」「订阅收入」等关键词
    - 有盈亏平衡分析

---

### 步骤 4：验证知识库命中效果

1. 在 AI 分析结果页面，展开「AI 完整推理」或「解释说明」部分
2. ✅ **验证点**：
   - 应该能看到「知识库命中结果」或类似描述
   - 显示命中的历史案例名称
   - 显示相似度百分比

3. 打开浏览器开发者工具（F12）
4. 切换到 Console 标签
5. 重新执行一次 AI 分析
6. ✅ **验证点**：
   - 应该能看到日志：`[AI 调参] 检索知识库...`
   - 应该能看到日志：`[AI 调参] 知识库命中：X 条`

---

### 步骤 5：对比测试（无知识库情况）

1. 在知识库面板中，删除所有上传的文档
2. 重新执行步骤 3 的测试用例 A
3. ✅ **验证点**：
   - 对比有/无知识库时的 AI 建议差异
   - 无知识库时，建议可能更通用、缺少行业针对性
   - 有知识库时，建议应该更具体、有历史案例参考

## 🐛 常见问题排查

### 问题 1：文档上传后搜索不到
**排查步骤**：
1. 打开浏览器 Console，查看是否有错误
2. 检查 `knowledgeService.initialize()` 是否成功
3. 确认文档内容已被正确解析

### 问题 2：AI 分析时没有知识库命中
**排查步骤**：
1. 确认知识库中有文档
2. 检查 Console 中的检索日志
3. 确认搜索词与文档内容有关联
4. 尝试调整相似度阈值（默认 0.3）

### 问题 3：AI 配置错误
**排查步骤**：
1. 点击「AI 决策」→「AI 配置」
2. 点击「测试连接」按钮
3. 确认 API URL 和 API Key 正确
