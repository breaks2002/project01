## 📋 AC/FC/BU 分布规则

```
时间轴：1 月 --- 8 月 --- 9 月 --- 12 月
        │实际期间│预测期间│

AC（实际）：只在已发生的期间存在（1-8 月）
FC（预测）：只在未发生的期间存在（9-12 月）
BU（目标）：全期间都有（1-12 月）
```

## 🔢 计算指标验证

### 毛利润计算验证

```
毛利润 = 营业收入 - 营业成本

202601: 1,250,000 - 780,000 = 470,000
202602: 1,380,000 - 850,000 = 530,000
...
```

### 营业利润计算验证

```
营业利润 = 毛利润 - 管理费用 - 销售费用

202601: 470,000 - 120,000 - 180,000 = 170,000
202602: 530,000 - 125,000 - 195,000 = 210,000
...
```

### 净利润计算验证

```
净利润 = 营业利润 × 0.75 (税率 25%)

202601: 170,000 × 0.75 = 127,500
202602: 210,000 × 0.75 = 157,500
...
```

## 📝 数据格式说明

### 新横表格式

```csv
指标 ID，指标名称，属性，202601,202602,202603,...
yingyeshouru，营业收入，AC,1250000,1380000,1420000,...
yingyeshouru，营业收入，FC,1650000,1750000,1820000,...
yingyeshouru，营业收入，BU,1800000,1900000,2000000,...
```

**特点**：
- 每个指标 **3 行**（AC/FC/BU 各一行）
- 时间列为字符串（支持跨年滚动）
- 属性列固定为 `AC` | `FC` | `BU`

### 公式表格式

```csv
指标 ID，指标名称，节点类型，公式，最小值，最大值，单位，显示格式，指标方向，层级，汇总方式，比率型指标
yingyeshouru，营业收入，driver,,800000,3000000，元，"#,##0",auto,1,sum,FALSE
maolirun，毛利润，computed,yingyeshouru - yingyechengben,,,元，"#,##0",auto,3,sum,FALSE
```
