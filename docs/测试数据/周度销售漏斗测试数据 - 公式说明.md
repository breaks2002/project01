## 🔧 PERIOD 函数测试公式

### 1. 基础汇总测试

```javascript
// 测试 1：汇总实际值 (AC)
PERIOD_SUM(访问流量，'AC')
// 预期：2026WK01-WK12 的访问流量总和

// 测试 2：汇总预测值 (FC)
PERIOD_SUM(访问流量，'FC')
// 预期：2026WK13-WK18 的访问流量总和

// 测试 3：汇总目标值 (BU)
PERIOD_SUM(访问流量，'BU')
// 预期：2026WK01-WK18 的预算总和

// 测试 4：默认汇总 (AC+FC)
PERIOD_SUM(访问流量)
// 预期：实际值 + 预测值（排除 BU）
```

### 2. 转化率计算测试

```javascript
// 测试 5：线索转化率 (平均)
线索转化率 = 线索数 / 访问流量
// 公式自动按月计算后取平均

// 测试 6：商机转化率 (平均)
商机转化率 = 商机数 / 线索数

// 测试 7：成交率 (平均)
成交率 = 成交额 / (商机数 × 1000)
```

### 3. PERIOD 函数高级用法

```javascript
// 测试 8：计算平均周访问量
PERIOD_AVG(访问流量)
// 预期：所有期间 (AC+FC) 的平均值

// 测试 9：计算实际期平均访问量
PERIOD_AVG(访问流量，'AC')
// 预期：WK01-WK12 的平均值

// 测试 10：找出最高周访问量
PERIOD_MAX(访问流量)
// 预期：所有期间中的最大值

// 测试 11：找出实际期最低访问量
PERIOD_MIN(访问流量，'AC')
// 预期：WK01-WK12 中的最小值

// 测试 12：统计有数据的周数
PERIOD_COUNT(访问流量)
// 预期：18 周 (AC 有 12 周，FC 有 6 周)

// 测试 13：统计非零周数
PERIOD_COUNT_NONZERO(成交额)
// 预期：18 周

// 测试 14：统计总期数 (包括空值)
PERIOD_COUNT_EXISTS(访问流量)
// 预期：19 周 (BU 有 19 周数据)
```

### 4. 混合表达式测试

```javascript
// 测试 15：全年总成交额 (汇总 AC+FC)
PERIOD_SUM(成交额)

// 测试 16：全年总成交额中实际占比
PERIOD_SUM(成交额，'AC') / PERIOD_SUM(成交额)

// 测试 17：平均周成交额
PERIOD_AVG(成交额)

// 测试 18：实际期平均周成交额
PERIOD_AVG(成交额，'AC')

// 测试 19：成交额目标完成率
PERIOD_SUM(成交额) / PERIOD_SUM(成交额，'BU')

// 测试 20：实际期目标完成率
PERIOD_SUM(成交额，'AC') / PERIOD_SUM(成交额，'BU', 'AC')
```

### 5. 嵌套公式测试

```javascript
// 测试 21：销售漏斗整体转化率
PERIOD_SUM(成交额) / PERIOD_SUM(访问流量) / 1000
// 预期：从访问到成交的整体转化率

// 测试 22：线索 - 商机转化率
PERIOD_SUM(商机数) / PERIOD_SUM(线索数)

// 测试 23：商机 - 成交转化率
PERIOD_SUM(成交额) / PERIOD_SUM(商机数) / 1000
```

### 6. 条件函数组合测试

```javascript
// 测试 24：如果成交额超过目标，显示"达标"，否则显示"未达标"
IF(PERIOD_SUM(成交额) >= PERIOD_SUM(成交额，'BU'), 1, 0)

// 测试 25：如果周均成交额超过 150 万，返回 1
IF(PERIOD_AVG(成交额) > 1500000, 1, 0)
```
