# AI 智能调参 - 重构使用说明

## 🎯 重构目标

解决原有架构的硬编码问题，支持多场景（财务/HR/生产等）的自适应 AI 调参。

---

## 📁 新增功能

### 1. Prompt 模板系统

#### 内置模板
| 模板名称 | 适用场景 | 关键词 |
|---------|---------|--------|
| 财务场景 | 成本优化、利润提升、收入增长 | 成本、利润、收入、费用 |
| HR 人力场景 | 招聘优化、人力成本控制、人效提升 | 招聘、人力、人效、流失 |
| 生产场景 | 产能提升、良率优化、生产计划 | 产能、良率、库存、设备 |
| 通用场景 | 默认模板，跨部门综合优化 | - |

#### 自定义模板
- ✅ 新建 Prompt 模板
- ✅ 编辑模板（内置模板创建副本）
- ✅ 删除模板（内置模板不可删除）
- ✅ 导入模板（支持 MD/JSON 格式）
- ✅ 导出模板（JSON 格式）
- ✅ 自动匹配（根据关键词）

---

### 2. 纯数据驱动 Prompt 组装

#### 架构设计（A+B+C 方案结合）

**方案 A**：动态 Prompt 生成
- aiPromptBuilder.js 只组装 JSON 数据
- 用户自定义的 prompt 作为 system prompt 传入

**方案 B**：Prompt 模板系统
- 多场景 Prompt 模板
- 自动匹配或用户选择

**方案 C**：纯数据驱动
- 只提供数据结构（模型、因子、分析数据）
- 用户业务背景直接传给 AI
- AI 自己决定如何调整

## 🚀 使用流程

### 步骤 1：配置 AI 参数
1. 点击 Toolbar「AI 决策」→「AI 配置」
2. 填写 API URL、API Key、模型名称
3. 点击「测试连接」验证
4. 保存配置

### 步骤 2：上传知识库文档（可选）
1. 点击 Toolbar「AI 决策」→「知识库」
2. 点击「➕ 上传文档」
3. 选择文件（支持 PDF/Word/Excel/TXT/MD）
4. 填写元数据（标题、行业、标签）
5. 确认上传

### 步骤 3：选择 Prompt 模板（可选）
1. 点击 Toolbar「AI 决策」→「AI 调参」
2. 点击右上角「⚙️」打开模板管理
3. 选择合适场景的模板，或导入自定义模板
4. 不选择则自动匹配场景

### 步骤 4：输入业务背景
```
示例（财务场景）：
我是一家制造企业，面临成本上涨压力，净利润率从 23% 下降到 18%。
希望优化成本结构，实现净利润率回升到 25%。
```

```
示例（HR 场景）：
公司研发部门扩编，计划 Q4 新增 50 名工程师。
目前招聘周期太长（平均 45 天），希望缩短到 30 天以内。
招聘预算可以适当增加，但要控制单人次招聘成本在 5000 元以内。
```

```
示例（生产场景）：
工厂目前产能利用率 75%，希望提升到 90%。
可以通过增加班次、设备升级、工艺优化来实现。
预算范围内，设备投资不超过 500 万。
```

### 步骤 5：执行 AI 分析
1. 点击「开始智能分析」
2. 等待 AI 分析完成（约 10-30 秒）
3. 查看调整建议
4. 可编辑调整值
5. 点击「一键应用全部」

## 📊 场景自动识别逻辑

系统根据用户输入的关键词自动匹配场景模板：

```javascript
// 财务场景关键词
['成本', '利润', '收入', '费用', '毛利', '净利', '财务']

// HR 场景关键词
['招聘', '人力', '人效', '流失', '薪酬', '培训', '人力成本']

// 生产场景关键词
['产能', '良率', '库存', '设备', '生产', '订单', '制造']
```

匹配规则：
- 匹配 2 个及以上关键词 → 对应场景
- 匹配少于 2 个 → 通用场景

## 📝 自定义 Prompt 模板格式

### Markdown 格式

```markdown
# 我的自定义场景

## 适用场景
- 场景 1 描述
- 场景 2 描述

## System Prompt

```
你是一位 XXX 专家...

【核心任务】
...

【强制要求】
...
```

```

### JSON 格式

```json
{
  "name": "我的模板",
  "description": "适用场景描述",
  "keywords": ["关键词 1", "关键词 2"],
  "systemPrompt": "System Prompt 内容"
}
```

## ⚠️ 注意事项

1. **内置模板不可删除**：财务、HR、生产、通用模板是内置的，不允许删除
2. **编辑内置模板会创建副本**：修改内置模板时，系统会自动创建自定义副本
3. **Prompt 模板与知识库独立**：Prompt 模板是分析逻辑，知识库是历史案例，两者配合使用
4. **场景匹配不中时**：如果自动匹配不准确，可以手动选择模板
