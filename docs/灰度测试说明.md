# AIDM v1.0.0 灰度测试说明

## 测试版本信息

- **版本号**: 1.0.0
- **打包日期**: 2026/06/01
- **安装包路径**: `release/AIDM Setup 1.0.0.exe`（135MB）
- **架构**: Windows x64
- **打包方式**: NSIS 安装程序

## 安装说明

1. 双击 `AIDM Setup 1.0.0.exe` 运行安装程序
2. 选择安装路径（默认即可）
3. 勾选创建桌面快捷方式
4. 点击安装，等待完成

## 测试重点

### 1. 授权模块

| 测试项 | 操作 | 预期结果 |
|--------|------|----------|
| 首次启动 | 打开软件 | 自动进入试用版，底部黄色状态条显示试用信息 |
| 导入授权 | 点击"升级专业版"→"导入授权文件" | 选择 `.lic` 文件后激活成功 |
| 试用版限制 | 点击 AI 决策、导出、PowerBI | 弹出提示"试用版不支持" |
| 试用版限制 | 新建节点超过 30 个 | 第 31 个被阻止 |
| 试用版限制 | 创建第 4 层节点 | 被阻止，提示"最多支持 3 层" |
| 安全加固 | 按 F12 / Ctrl+Shift+I | 不弹出 DevTools |
| 安全加固 | 右键点击界面 | 不弹出菜单 |
| 授权篡改 | 手动修改 license.lic 内容 | 重新打开软件提示"授权无效" |

### 2. 核心功能

| 测试项 | 操作 | 预期结果 |
|--------|------|----------|
| 新建节点 | 点击"新建节点" | 编辑器正常打开 |
| 导入数据 | 点击"导入数据" | CSV/JSON 导入正常 |
| 方案比选 | 创建多个方案后点击"方案比选" | 面板正常显示 |
| 敏感性分析 | 选择节点后点击"敏感性分析" | 图表正常渲染 |
| 标准差分析 | 点击"标准差分析" | 面板正常显示 |
| 趋势图 | 点击节点卡上的"查看趋势图" | 图表正常显示 |
| 画布缩放 | 默认进入 | 缩放比例 60%，不会太大 |

### 3. PBI 连接（如测试环境有 PowerBI）

| 测试项 | 操作 | 预期结果 |
|--------|------|----------|
| PBI 连接 | 点击"PowerBI"按钮 | 连接弹窗正常显示 |
| 数据刷新 | 点击"刷新"按钮 | 数据从 PBI 正常拉取 |
